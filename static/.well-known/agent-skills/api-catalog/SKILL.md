# API Catalog Skill

This site publishes an API catalog at `/.well-known/api-catalog` per RFC 9727.

## Catalog Location

`GET /.well-known/api-catalog`
`Content-Type: application/linkset+json`

The catalog includes link relations for site documentation and blog feed endpoints.

## Reference

- RFC 9727: https://www.rfc-editor.org/rfc/rfc9727
- RFC 9264: https://www.rfc-editor.org/rfc/rfc9264
