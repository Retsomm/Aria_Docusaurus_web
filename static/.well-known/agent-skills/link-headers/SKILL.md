# Link Headers Skill

This site includes `Link` response headers on all pages per RFC 8288 to advertise resources useful for agent discovery.

## Headers Added

- `rel="api-catalog"` → `/.well-known/api-catalog` (RFC 9727 API catalog)
- `rel="service-doc"` → `/docs/intro` (site documentation)

## Reference

- RFC 8288: https://www.rfc-editor.org/rfc/rfc8288
- IANA Link Relations: https://www.iana.org/assignments/link-relations/
