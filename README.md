# 水星領航員，火星上的新威尼斯

這是一個介紹動畫《水星領航員》（ARIA）與火星新威尼斯世界觀的單頁式網站，包含故事背景、角色介紹、未來展望等內容。

## 專案結構

```
index.html      # 主網頁
style.css       # 網站樣式
script.js       # 網站互動腳本
```

## 使用技術

- HTML5
- CSS3
- JavaScript (jQuery, Skrollr)
- Bootstrap 3
- Font Awesome

## 如何使用

1. 下載或 clone 此專案
2. 於瀏覽器中開啟 `index.html` 即可瀏覽網站

## 特色

- 響應式導覽列與動畫效果
- 介紹火星新威尼斯的世界觀與主要角色
- 支援 Skrollr 捲動動畫
- 現代化設計與視覺效果
