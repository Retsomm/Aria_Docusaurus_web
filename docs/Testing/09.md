# Chromium 是什麼？淺白介紹與生活應用

## 一、先講個你一定懂的比喻

你有沒有用過 Google Chrome 瀏覽器？

如果有，那你已經用過 Chromium 了，只是你不知道而已。

簡單說：

> **Chromium 是一套「免費、公開」的瀏覽器引擎，Google Chrome 是用 Chromium 做出來的「商品版」。**

就像麵包店把麵粉、酵母這些「公開配方」拿來，加上自己的招牌奶油、包裝，變成一個品牌麵包來賣。Chromium 就是那個公開配方，Chrome 是加了 Google 招牌料的成品。

## 二、Chromium 到底是什麼？

Chromium 是一個**開放原始碼（open source）**的瀏覽器專案，主要由 Google 主導開發，但任何人都可以看到它的程式碼、修改它、甚至拿去做出自己的瀏覽器。

它包含了瀏覽器最核心的東西：

- **網頁排版引擎（Blink）**：負責把 HTML、CSS 變成你看到的網頁畫面
- **JavaScript 引擎（V8）**：負責執行網頁上的程式邏輯，讓網頁能互動
- **使用者介面框架**：分頁、網址列、書籤等基本介面元件
- **安全機制**：例如「多工作程序（multi-process）」設計，讓每個分頁獨立運作，一個分頁當掉不會拖垮整個瀏覽器

你可以把它想成「蓋房子用的鋼筋、水泥、地基」，房子要怎麼裝潰、漆什麼顏色，就是各家廠商自己決定。

## 三、為什麼會有 Chromium？

時間回到 2008 年，Google 推出了自家瀏覽器 Chrome。當時 Google 做了一個特別的決定：把 Chrome 大部分的原始碼公開出來，變成一個獨立的開源專案，這就是 Chromium。

這樣做的好處是：

1. **大家都能參與改進**：全世界的工程師都能貢獻程式碼、抓漏洞
2. **其他公司能直接拿來用**：不用從零開始寫一套瀏覽器引擎
3. **網頁標準更統一**：大家用同一套引擎，網頁相容性問題變少

正因為這樣開放，Chromium 才能在十幾年內變成全球瀏覽器市場的「地基」。

## 四、誰在用 Chromium？比你想的還多

你可能以為只有 Chrome 在用，其實一大堆你天天在用的瀏覽器，骨子裡都是 Chromium：

| 瀏覽器 | 說明 |
|---|---|
| **Google Chrome** | Chromium 加上 Google 自家功能（密碼管理、帳號同步、影音解碼器等） |
| **Microsoft Edge** | 微軟在 2020 年放棄自家引擎，改用 Chromium 打造新版 Edge |
| **Brave** | 主打隱私保護、預設封鎖廣告與追蹤器 |
| **Opera** | 老牌瀏覽器，現在也是 Chromium 基礎 |
| **Vivaldi** | 主打高度自訂化，分頁管理功能特別強 |
| **三星瀏覽器（Samsung Internet）** | 三星手機內建的瀏覽器 |
| **Android WebView** | 許多手機 App 內嵌的「小型瀏覽器」，例如 App 裡點連結跳出的內頁，背後常常就是它 |

換句話說，全世界桌面電腦有超過七成的網頁瀏覽流量，都是透過 Chromium 系的瀏覽器在跑。

## 五、Chromium 和 Chrome 差在哪？

很多人搞不清楚這兩個的差別，簡單列出來：

| | Chromium | Google Chrome |
|---|---|---|
| 是否開源 | 是，原始碼公開 | 否，是 Google 的商業產品 |
| 自動更新 | 沒有內建自動更新機制 | 有，會自動幫你更新 |
| 影音格式支援 | 較少（需自行安裝外掛） | 內建較完整（如 H.264 等付費授權格式） |
| 帳號同步 | 沒有 Google 帳號整合 | 可同步書籤、密碼、瀏覽紀錄 |
| 適合對象 | 開發者、注重隱私與客製化的人 | 一般大眾使用者 |

簡單講：**Chromium 是「半成品」，Chrome 是「加值後的成品」**。一般人日常使用，直接用 Chrome 即可；如果你在意隱私、想要更純淨沒有 Google 服務介入的版本，才會特別去找 Chromium 來用。

## 六、Chromium 的應用不只瀏覽器

除了拿來做瀏覽器，Chromium 還有一個很厲害的延伸應用，叫做 **Electron**。

Electron 讓開發者可以用網頁技術（HTML、CSS、JavaScript）做出**桌面應用程式**，背後其實是包了一個 Chromium 在裡面執行畫面。

你可能每天都在用 Electron 做的軟體，例如：

- **Slack**（團隊溝通軟體）
- 許多筆記、聊天、開發工具類 App

也就是說，Chromium 早就不只是「瀏覽器的骨架」，它也是現代很多桌面軟體背後的隱藏功臣。

## 七、給初學者的重點整理

1. Chromium 是 Google 主導的**開源瀏覽器專案**，任何人都能取得程式碼
2. Chrome、Edge、Brave、Opera、Vivaldi 等知名瀏覽器，**都是站在 Chromium 這個地基上**蓋出來的
3. Chromium 提供「裸版」功能，各家公司再加上自己的招牌特色（隱私、自訂化、帳號整合等）
4. Chromium 的影響力不只在瀏覽器，連 Slack 這類桌面軟體背後也藏著它（透過 Electron）
5. 因為是開源授權，就算 Google 有一天不再維護，其他公司或社群仍可以接手延續下去

## 八、一句話總結

**Chromium 就是現在整個網路世界，幾乎所有瀏覽器背後共用的「公共引擎」。**

下次你打開瀏覽器上網時，可以想一下：不管你用的是 Chrome、Edge 還是 Brave，畫面背後可能都跑著同一套源自 Chromium 的引擎。
